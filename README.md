# Music Streaming SQL Project

This SQL group project analyzes a dataset of 500 songs and 100 users. 
It includes:
- Table creation and data insertion scripts
- Queries to identify highest-rated songs and most active listeners
- Sample analysis of content popularity and listening behavior

Database Schema Overview:
Artist: Stores individual artists and groups
Genre: Song genres
Album: Albums linked to artists and genres
Song: Songs linked to artists and albums
Song_genre: Many-to-many relationships between songs and genres
User: Users of the streaming platform
Rating: User ratings for songs
